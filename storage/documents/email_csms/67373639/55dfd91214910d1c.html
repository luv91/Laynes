CSMS # 67373639 - Updates to Submitting HTS Codes when Filing an Air Import Manifest Available in PROD

U.S. Customs and Border Protection (CBP) has released into Production (PROD) Automated Commercial Environment (ACE) an update to the Declared Value and ISO Currency Code fields to be C for Conditional, required for an express bill.

CBP has updated the ACE AIR CAMIR Message Line Identifiers Implementation Guide: https://www.cbp.gov/document/guidance/ace-air-camir-message-line-identifiers

Please reach out to your Client Representative if you have any technical questions. If you have any questions about this message, please contact CREM@cbp.dhs.gov.
